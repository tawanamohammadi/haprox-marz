Quick Install

git clone https://github.com/tawanamohammadi/haprox-marz.git
cd haprox-marz
chmod +x install.sh
sudo ./install.sh
1Install HAProxy SNI router
2Install Cloudflare Warp for Marzban
3Full setup (both)
